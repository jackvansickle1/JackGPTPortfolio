01 / CLI docs

Spreadsheet Rescue

A dependency-free Python CLI documented around a conservative data-cleaning promise: the tool should never guess, overwrite the input, or hide what changed.

README / quickstart / safety model

Make the first run reversible.

The public guide starts with a zero-write dry run, then shows the production command and the three files a user should expect.

  • PowerShell-first installation and invocation
  • Explicit configuration scope for typed fields
  • No-overwrite and ambiguous-value guarantees
  • Machine-readable and human-readable audit outputs
  • Sixteen automated tests behind the public release
